# Role: Client

## Description
The client makes requests to the server. Here are the requests that the client should make.

1. GET /people/1
2. GET /things/3
3. POST /places ({name: CN Tower, country: Canada, latitude: 43.642566, longitude: -79.387057})
4. PATCH /things/8 ({price: 6.99})
5. GET /people
6. DELETE /places/7
7. PATCH /things/20
